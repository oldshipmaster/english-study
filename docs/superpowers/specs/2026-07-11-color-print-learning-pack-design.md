# StoryStage 彩色纸质故事学习包设计

## 1. 目标

把 StoryStage 从“可打印剧本”升级为以彩色 A4 纸质练习为核心的家庭英语学习包。每次练习控制在 25–30 分钟，通过同一个故事完成词汇预习、句型理解、分角色开口、纸上标记、演后复习和主动记忆。

主要使用者是 9–11 岁、具备小学基础英语的女儿，以及 1–2 位陪练家长。网站继续负责选择故事、分配角色和生成打印材料；实际学习主要发生在纸上。

## 2. 学习闭环

每个故事学习包包含八个连续环节：

1. 家庭任务卡：角色、时间安排、本次要掌握的词和句型。
2. 演前词汇热身：看词猜义、遮住中文回忆、勾选“我认识 / 有点难 / 我不会”。
3. 彩色完整剧本：保留全剧台词，用角色颜色、图标和姓名共同区分。
4. 女儿角色练习页：集中她的台词，提供重音、停顿和情绪提示。
5. 句型与语法发现页：从故事抽取两个核心句型，用替换槽理解语法，不使用抽象术语堆砌。
6. 开口挑战页：脱离完整台词完成提示说句、替换造句和家庭即兴对话。
7. 可裁剪记忆卡：词汇卡与句型卡，供女儿保留并主动复习。
8. 家长抽查页：答案、口头提问顺序、第二天与一周后的复习建议。

## 3. 主动记忆机制

纸面上的每个重点词均有三个掌握状态：绿色“我认识”、黄色“有点难”、红色“我不会”。女儿必须自己勾选，家长不代替判断。所有黄色和红色项目进入可裁剪卡片复习。

每张词汇卡正面包含英文、音节或重音提示和小图标；背面包含中文、故事原句和一个可替换例句。句型卡正面是带空槽的英文框架，背面是中文意图和两个替换示例。

家长抽查采用“看中文说英文 → 补完整句 → 换一个词造新句”的三级提示。答不出时先给首字母，再给故事情境，最后才展示答案。

## 4. 词汇与句型选择

每个故事选 8 个重点词，其中 5 个当前故事新词、3 个来自以前故事的滚动复现词。首版六个故事没有学习历史，因此按固定故事顺序设置复现关系；单独打印任意故事时仍可完整使用。

每个故事选 2 个核心句型。每个句型包含：用途中文说明、故事原句、结构化模板、3 个替换词、2 个口头造句任务和一条轻量语法提醒。语法提醒使用儿童可理解的表达，例如“一个人正在做什么：人物 + is + 动词-ing”，不使用长篇规则讲解。

## 5. 彩色 A4 版式

学习包使用 A4 纵向布局，彩色但兼容家庭打印：奶油底、墨蓝正文、蜂蜜黄任务、薄荷绿已掌握、珊瑚红待复习、淡紫句型。大面积背景保持浅色以节省墨水。

每个打印页面都有故事名、页型、页码和裁切安全边距。内容块禁止跨页断裂。卡片使用虚线裁切框，单页最多 8 张词汇卡或 4 张句型卡。家长答案页明确标注“家长保管”。

## 6. 数据与组件

故事数据新增 `learningPack`，包含目标词汇、复现词汇、两个句型和口头挑战。现有 `Story`、台词和挑战数据保持兼容。

- `LearningPackPrint`：生成整套八环节纸质材料。
- `VocabularyWarmup`：打印词汇掌握状态表。
- `SentencePatternSheet`：生成句型发现与替换练习。
- `MemoryCards`：生成可裁剪词汇卡和句型卡。
- `ParentCoachSheet`：生成答案与间隔复习脚本。

打印入口提供“彩色故事学习包”作为推荐选项，同时保留原有“完整家庭剧本”和“角色专属剧本”。

## 7. 错误与降级

缺少学习包数据的故事仍可打印原有剧本，并显示“该故事暂未配置学习包”。浏览器打印失败时继续展示浏览器菜单打印说明。颜色不是唯一信息通道，状态同时使用文字、符号和边框。

## 8. 验收

- 六个故事均拥有 8 个重点词、2 个句型和完整学习包。
- 打印入口能选择学习包、完整剧本或角色专属剧本。
- 学习包包含八个规定环节，角色与女儿台词对应当前分配。
- 所有重点词有三态纸面标记和可裁剪卡片。
- 每个句型有原句、模板、替换词、语法提醒和口头任务。
- 家长页包含答案、三级提示和次日/一周复习安排。
- 打印 CSS 提供 A4、分页、页码、裁切线和节墨配色。
- 现有在线排练、挑战、GitHub Pages 构建和部署流程不受影响。
